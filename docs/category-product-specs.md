# Loại sản phẩm
## 1. Tai nghe In-Ear (IEMs / Earbuds)
Đây là dòng tai nghe nhỏ gọn, nhét sâu vào ống tai hoặc đặt ở cửa tai.

Specs đặc thù:

Driver Configuration: Cấu hình driver (ví dụ: 1 Dynamic + 2 Balanced Armature).

Eartips Type: Chất liệu (Silicone, Foam), số lượng size đi kèm.

Nozzle Diameter: Đường kính ống dẫn âm (quan trọng để thay thế tips).

Fit Type: Universal hay Custom-fit.

## 2. Tai nghe Over-Ear / On-Ear (Fullsize)
Dòng tai nghe có gọng (headband), ôm trọn hoặc nằm trên vành tai.

Specs đặc thù:

Acoustic Design: Open-back (âm mở) hay Closed-back (âm đóng).

Ear-pad Material: Chất liệu đệm tai (Da cừu, Nhung, Da PU).

Clamp Force: Lực ép của gọng (thường tính bằng Newton).

Weight: Trọng lượng (quan trọng đối với sự thoải mái khi đeo lâu).

## 3. Thiết bị giải mã & Khuếch đại (DAC/AMP)
Phụ kiện bổ trợ để xử lý tín hiệu và tăng công suất cho tai nghe.

Specs đặc thù:

Output Power: Công suất đầu ra (mW @ Ohm).

DAC Chip: Tên chip xử lý (ví dụ: ESS Sabre, AKM).

Sample Rate: Tốc độ lấy mẫu tối đa (ví dụ: 32-bit/768kHz, DSD512).

Inputs/Outputs: Các cổng kết nối (3.5mm, 4.4mm Balanced, USB-C, Optical).

## 4. Loa Bluetooth / Loa kiểm âm (Speakers)
Specs đặc thù:

Power Output (RMS): Tổng công suất thực.

Frequency Response: Dải tần đáp ứng (thường sâu hơn tai nghe).

Battery Life: Thời lượng pin (đối với loa di động).

Water Resistance: Chỉ số chống nước (IPX7, IP67).

# Bảng tổng hợp Specs chung (Base Specs)
Dù là loại nào, tất cả sản phẩm audio của bạn đều nên có các thông số nền tảng sau trong productTable:

| Thông số            | Ý nghĩa                              | Đơn vị                        |
| ------------------- | ------------------------------------ | ----------------------------- |
| Impedance           | Trở kháng                            | Ohm (Ω)                       |
| Sensitivity         | Độ nhạy                              | dB/mW hoặc dB/V               |
| Frequency Response  | Dải tần (Min - Max)                  | Hz - kHz                      |
| Driver Type         | Loại driver (Dynamic, Planar, BA)    | Enum                          |
| Connector Type      | Chuẩn cắm (2-pin, MMCX, Fixed)       | String                        |
